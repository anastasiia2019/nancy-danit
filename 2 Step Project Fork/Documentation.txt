Методология и Правила написания структуры документа веб-страницы.	
Название изменения	Описание изменения.
	
1) Шрифт. 	 Прописываем общий в боди
2) Ограничение контента по странице и центрирование. 	 Создаем сontainer{с max-width!}
3) Название переменных. 	Без нижчнего подчеркивания. Используем тире!
4) Структура секций. 	В боди делаем секции (header, section, footer)
5) Меню 	Делаем внутри секции header меню структурой ul>li>a- это важно для backend языка. Класс меню назывем (navbar-menu// каждый элемент которого называем navbar-menu-item,внутри каждого элемента будет ссылка через тег a (navbar-menu-link)
6) Логотип	 Внутри секции header делаем navbar внутри которого будет логотип выполнен ссылкой через тег а..(navbar-logo/  navbar-logo/ navbar-logo-img/  navbar-logo-text/  ). Ссылка логотипа скорре всего не должна ссылаться сама на себя на главной странице. ??? Нужно уточнить. 
6) Кнопки. 	У нас они одинаковые по цвету фона(который определяет семейство шрифта для текста внутри кнопки, толщину рамки, значение свойства display и закругление краев)  и размеру с цветом текста,но разные по размеру(font-size и padding). Соответсвенно даем универсальный класс (btn и btn--small и btn--large). Еще есть кнопка в последней секции по функции (btn-purchase).  
7) Название секции.	 (section-title). и вторым классом-модификаиором будет по размеру section-title-size
8) Описание под названием секции 	 (section-description) и вторым классом-модификаиором будет по размеру section--description-size
9) img	Для картинок оставляем тег alt. Для заполения Seo спец
10) Лайк со счетчиком	
11) Ссылки	"прописываем общий для всех чтобы убрать подчеркивание     a {
    text-decoration: none;
}"
12) Пункты меню 	"прописываем общий для всех чтобы убрать точки ul {
  list-style: none; }
"
13) Курсор мышки 	"Общие для всего тела button {
  cursor: pointer; }
